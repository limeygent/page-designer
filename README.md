# Page Designer Skill

Layout-first visual design for Bootstrap 5 service pages. Thinks like a designer — assesses the page's purpose, feels the creative direction, and draws from a toolkit of design principles, treatments, and components to produce pages that look custom-designed rather than AI-generated.

## Quick Start

```
/page-designer
```

**Minimum inputs:**
- Keyword: "emergency plumber Dallas"
- Niche: plumbing
- Brand color: #1B6AAA

## What It Does

1. **Understands the page** — who is arriving, what they're feeling, what the page needs to communicate
2. **Designs the layout** — chooses visual treatments, component variants, and applies the design principles that matter most for this specific page
3. **Refines and reviews** — scans for mass-production signals, checks visual hierarchy, ensures the page flows naturally

## How It Fits With Other Skills

```
Intent Analyzer → Service-Page (section ordering + content strategy)
                        ↓
                 Page Designer (visual treatments + layout)
                        ↓
                 Service-Page / Content-Council (content fills the layout)
```

The page-designer does NOT override the service-page skill's section ordering.

## Key Concepts

- **7 Section Treatments** (T1-T7): Open Canvas, Tinted Band, Card Grid, Split Panel, Accent Band, Boxed Container, Full-Width Image Hero — a visual palette to choose from
- **Component Variants**: 3-4 visual options for each content type (lists, data, testimonials, CTAs, steps)
- **13 Design Principles Toolkit**: Visual hierarchy, section rhythm, spatial awareness, attention anchoring, symmetry/asymmetry, reading patterns, visual grids, color as information, typography rhythm, progressive disclosure, Gestalt principles, consistency, Fitts's Law — lenses to look through when making decisions, not a checklist to apply sequentially
- **9 Anti-Pattern Signals**: H2 Parade, White Wall, Card Carpet, Icon Assembly Line, Symmetry Overload, Table Tunnel, CTA Echo, Spacing Monotony, Anchor Collision — things to notice, not automatic failures
